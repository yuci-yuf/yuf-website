# YUF 2025 videos — drop your files here

The YUF 2025 page auto-plays these on scroll (muted, looping; they pause when
scrolled out of view, and viewers can pause/resume by clicking). Add the files
with these **exact names**:

| File | Where it plays |
|------|----------------|
| `event-highlights.mp4` | Event Highlights section — the festival film |
| `sukhee-lee.mp4` | Chief Guests section — Prof. Sukhee Lee's spotlight |

Recommended format: **H.264 video + AAC audio, `.mp4`**, ~720p–1080p, a few MB.

Until a file is present, that section shows its poster image with a
"Video coming soon" badge — nothing breaks. Drop the file in and it just works.
